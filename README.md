# Dataset Generator Pro

## Overview
Dataset Generator Pro is a Flask-based web application that generates realistic retail datasets for Data Analysis, Data Science, SQL, and Machine Learning practice.

## Features
- Clean Dataset Generation
- Dirty Dataset Generation
- Error Intensity Control
- CSV Export
- Excel Export
- Column Selection
- Download Clean Solution Dataset

## Technologies Used
- Python
- Flask
- Pandas
- HTML
- CSS

## Dataset Columns
- Order ID
- Order Date
- Ship Date
- Customer ID
- Customer Name
- ...
- Reorder Quantity

## Supported Data Quality Errors
- Missing Values
- Blank Values
- Duplicate Records
- Duplicate Customer IDs
- Outliers
- Negative Values
- Typographical Errors
- Inconsistent Categories

## Project Workflow
User Input → Dataset Generation → Error Injection → File Export → Download
